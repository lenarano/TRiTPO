Руководство пользователя

Zipper-инструмент для архивирования и управления архивами. Архиватор совмещает возможности работы как с графическим интерфейсом пользователя(GUI), так и консольной версией(CLI), запускаемой из командной строки и работающей в текстовом режиме.

Некоторые отличительные особенности Zipper:

  • оригинальный алгоритм сжатия данных без потерь;
  • поддержка склеивания нескольких файлов в один для их последующей архивации в один файл;
  • наличие других дополнительных функций, например, самостоятельное определение папки-назначения для архивного файла и разархивированного файла, возможность самостоятельно определить "имя" для архива, установка пароля на архив и пр.

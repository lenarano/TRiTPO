Консольная версия(CLI)

Процесс использования приложения в консольном режиме предназначен для продвинутых пользователей.

  • Для запуска приложения в режиме консольного использования необходимо первым параметром запуска передать «-no-gui».
  • Параметр «-pack» указывает на необходимость упаковки (compress), параметр «-unpack» - на необходимость распаковки (decompress).
  • После следующего параметра «-files» прописывается путь к файлу(ам).
  • После «-path» указывается путь, по которому будет распологаться файл(лы).
  • Параметр «-name» используется при сжатии и позволяет указать имя для файла-архива.
  • Последний параметр «-password» позволяет указать пароль для файла-архива.

    • Примеры задания параметров:

    Архивация-no-gui -pack -files C:\test\text1.txt C:\test\text2.txt C:\test\text3.txt -path C:\test -name proba -password пароль

    Разархивация-no-gui -unpack -files C:\test\proba.bin -path C:\test\unpack -password пароль
